# Screeny aplikacji

W tym folderze należy umieścić zrzuty ekranu aplikacji:

- ekran tworzenia trasy,
- ekran mapy z punktami,
- ekran listy tras.

W środowisku automatycznym bez możliwości uruchomienia bundlera Expo i emulatora
nie udało się wygenerować prawdziwych screenshotów. Po uruchomieniu lokalnym uzupełnij
folder o pliki PNG/JPG.
